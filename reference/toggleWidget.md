# Hide/show a widget w.r.t a condition.

Wrapper for the toggleWidget function of the package \`shinyjs\`

## Usage

``` r
toggleWidget(widget, condition)
```

## Arguments

- widget:

  The id of a \`Shiny\` widget

- condition:

  A \`logical(1)\` to hide/show the widget.

## Value

NA
