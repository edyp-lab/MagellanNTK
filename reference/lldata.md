# lldata dataset

An instance of the class \`MultiAssayExperiment\` which contains one
assay (instance of class \`SummarizedExperiment\`): 'Convert': Contains
a data.frame with 6 columns and 100 rows fullfilled with '0'. It also
contains a slot 'history' in the metadata(). This dataset is the most
bacis one to run the example pipeline and learn how to work with
MagellanNTK

## Usage

``` r
data(lldata)
```

## Format

An object of class \`MultiAssayExperiments\`
