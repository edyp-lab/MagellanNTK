# Layout for workflows

Creates a layout to place the sidebar and the main screen for pipelines
and processes.

## Usage

``` r
process_layout(session, ns, sidebar, content)

process_layout_process(session, ns, sidebar, content)

process_layout_pipeline(session, ns, sidebar, content)
```

## Arguments

- session:

  The R session parameter

- ns:

  Namespaced IDs for inputs/outputs

- sidebar:

  Shiny components

- content:

  Shiny components

## Value

A \`div()\` containing shiny components
