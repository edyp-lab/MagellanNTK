# Get the version of a package installed in the local R distribution

Get the version of a package installed in the local R distribution

## Usage

``` r
GetPackageVersion(pkg)
```

## Arguments

- pkg:

  The name of a package

## Value

A \`character()\`

## Examples

``` r
GetPackageVersion('MagellanNTK')
#> [1] "0.99.29"
```
