# Get the first mandatory step not validated

Get the first mandatory step not validated

## Usage

``` r
GetFirstMandatoryNotValidated(range, rv)
```

## Arguments

- range:

  xxx

- rv:

  A \`list\` with at least two slots : \* mandatory: xxx \*
  steps.status: xxx

## Value

An integer which is the indice of the identified step in the vector
rv\$steps.status
